# Aplikasi Manajemen Tugas Kelompok

## Deskripsi
Website sederhana untuk mencatat dan memantau tugas kelompok dalam pengerjaan mini project. Aplikasi ini membantu mencatat tugas, anggota, status, dan deadline secara terpusat, lengkap dengan fitur tambah, edit, dan hapus tugas.

## Fitur
- Menampilkan daftar tugas kelompok
- Pencarian tugas berdasarkan judul atau anggota
- Filter tugas berdasarkan status (Belum Dikerjakan, Proses, Selesai)
- Filter tugas berdasarkan prioritas (Rendah, Sedang, Tinggi)
- Tambah tugas baru dengan validasi input
- Edit tugas yang sudah ada
- Hapus tugas (dengan konfirmasi)
- Ringkasan jumlah tugas dan progress kelompok di halaman Beranda

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
- JSON (data lokal)
- Git/GitHub

## Cara Menjalankan
1. Clone repository ini.
2. Buka folder proyek di VS Code.
3. Install extension "Live Server" (jika belum ada).
4. Klik kanan pada `index.html` → pilih "Open with Live Server".
5. Website akan terbuka otomatis di browser.
